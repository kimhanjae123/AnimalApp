# 프로젝트 소개
- **아나모네**란 **희망,기대**라는 꽃말을 가지고 있습니다.
- **아나모네**는 전국에 분포해있는 유기동물 보호소에 대한 정보를 제공하는 사이트입니다.
- **아나모네**에서 가까운 보호소를 확인하여 봉사활동도 참여할 수 있습니다.
- **아나모네**에서 많은 유기동물에게 **희망과기대**를 주세요.
